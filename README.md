# Astroneer Guide
Astroneer Guide is a simple guide to the planets and resources of the game [Astroneer](https://astroneer.space/) by [System Era](https://www.systemera.net/).

## Features
Astroneer Guide features:
 - A Complete Resource Menu: View How to find and make natural, atmospheric, refined, and composite resources
 - Planet Cards: Learn about different aspects of each of the 7 planets 
 - Clean UI: Smooth transitions and nice hover effects
 - Constellation Maker: A fun constellation maker in the background that slowly fades away. Connect stars by hovering over them.
